• ? : (question mark): Matches zero or one occurrence of the preceding character.
  • * : (asterisk): Matches zero or more occurrences of the preceding character.
  • + : (plus sign): Matches one or more occurrences of the preceding character.
  • . : (dot): Matches any single character except a newline. and that character can be uppercase, lowercase, digits, special symbols.
  • ^ : (caret): Matches the beginning of the string.
  • $ : (dollar sign): Matches the end of the string.
  • | : (pipe): Represents an OR operator, matching either the expression before or after it.
  • () : (parentheses): Groups parts of the expression together.
  • [] : (square brackets): Defines a character set.
  • \ : (backslash): Escapes special characters, allowing them to be treated as literal characters.
